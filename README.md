# Raisa-Coding-Challenge

### create new working environment (python=3.6+)
```bash
virtualenv -p YOUR/PATH/TO/PYTHON36 env36
```
### activate env
```bash
env36\Scripts\activate
```
### install requirments
```bash
pip install -r requirments.txt
```
### run unittests
```bash
cd coding_challenge
python -m unittest -v
```
